Submit a Design

  1. Home
  2. Submit a Design

Files can be submitted online or in person. The Cube is staffed by students very experienced with 3D printing and can answer any questions about the printing process.

Each printer in the cube has a unique software used to submit prints. To use our Dremel Printers you will have to visit the cube and attend a class that will get you setup with an account to use 3DprinterOS which is a software that allows you to submit prints remotely.

For our other printers you will have to fill out a submission form and email thecube@eng.ua.edu with your .stl file.

When submitting a part, please include:

  • Name
  • CWID
  • College and department
  • File name
  • Short description
  • General dimensions and scale
  • Purpose of part such as personal, research, etc.
  • How part will be used such as display for presentation, rocket fin, camera mount, etc.
  • Number of copies requested
  • Printer requested
  • Other important information

Original design files submitted should be .STL file to be used by the printers or .WRL file if full color is needed. The Cube staff can help generate these files.

The Cube reserves the right to refuse printing questionable objects including, but not limited to, weaponry, copyrighted materials or parts intended for commercial sale.
